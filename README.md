# sakib's dotfiles
setting up bash, vim, git

How to:
```
$ git clone https://github.com/sakib/dotfiles
$ cd dotfiles
$ cp .* ~
```

Install figlet, fortune-mod, cowsay and lolcat gem

Set up Sublime vimrc:
  https://github.com/grigio/vim-sublime

Finally, source your bashrc:
```
$ source ~/.bashrc
```
